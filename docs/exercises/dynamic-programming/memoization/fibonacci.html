Fibonacci¶

The Fibonacci sequence is 1, 1, 2, 3, 5, 8, .... Each successive number is found by adding the previous two. Create a recursive algorithm to compute the nth number in the Fibonacci sequence. Use memoization to get the last test to pass in a respectable about of time.

fib(1) -> 1
fib(2) -> 1
fib(3) -> 2
fib(4) -> 3
fib(5) -> 5
